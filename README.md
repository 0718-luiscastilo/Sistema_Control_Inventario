# Sistema_Control_Inventario
Sistema de Control de Inventario
  
Descripción

Aplicación de consola desarrollada en C# para gestionar el inventario de una tienda de tecnología.

El sistema permite registrar productos, consultar existencias, identificar productos agotados o con bajo stock y generar reportes básicos para apoyar la administración del inventario.

Funcionalidades
  
  Registro de productos  
  Consulta de todos los productos  
  Búsqueda de productos por nombre  
  Detección de productos con bajo inventario  
  Detección de productos agotados  
  Cálculo del valor total del inventario  
  Identificación del producto más caro  
  Identificación del producto con mayor cantidad disponible  
  Validación de entradas del usuario   
  Menú interactivo
